页面布局的本质：

  • 盒子模型
  • 浮动
  • 定位

盒子模型

css盒子模型本质是一个盒子，封装周围的元素，它包括边框(border)、外边距(margin)、内边距(padding)、内容

盒子实际长度 = width(内容) + 左右边框 + 左右内边距

盒子实际高度 = height(内容) + 上下边框 + 上下内边距

边框属性
  • 宽度border-width: 30px
  • 样式border-style: solid dashed dotted
  • 颜色border-color
  • 合并相邻边框 borer-collapse: collpase
简写 border: 1px red solid 没有顺序

只设置一条： border-top: 1px solid red

会影响盒子的大小

内边距

内容距离边框的距离

属性值从上开始，顺时针, 可以有1~4个值

会影响盒子的大小

外边距

两个盒子之间的距离

属性值从上开始，顺时针, 可以有1~4个值

外边距应用：把块级盒子水平居中，满足两个条件即可
1.盒子必须指定宽度
2.margin:0 auto;

如何让行内元素或行内块居中? 将父元素text-align:center(对于父元素而言，子行内或子行内块就像一个普通的文本)

嵌套块元素垂直外边距塌陷

问题：对于两个嵌套的块元素，父元素和子元素同时有外边距的时候，父元素会塌陷较大的边距值
解决方案：设置父元素的外边距或内边距，或者overflow：hidden

清除内外边距

浏览器默认会有内外边距，所以开发中一般在css第一行会清除内外边距

* {margin：0；padding 0}
