新增选择器

属性选择器 权重是10

  • 具有value属性的标签 x[value]
  • value属性是hello的标签 x[value="hello"]
  • value属性以love开头的标签 x[value^="love"]
  • value属性以end结尾的标签 x[value$="end"]
  • value属性含有li的标签 x[value*="li"]

结构伪类选择器 权重是12

  • ul li:first-child
  • ul li:last-child
  • ul li:nth-child(2)
  • ul li:nth-child(even) 偶数
  • ul li:nth-child(odd) 奇数
  • ul li:nth-child(n) 所有相当于循环n次 n叠加 必须是n字母
  • ul li:nth-child(公式n) 2n 或 n+2等等
  • ul li:first-of-type
  • ul li:last-of-type
  • ul li:nth-of-type(2)
  • ul li:nth-of-type(even) 偶数
  • ul li:nth-of-type(odd) 奇数
  • ul li:nth-of-type(n) 所有相当于循环n次 n叠加 必须是n字母
:nth-child和nth-of-type区别在于：前者先匹配冒号后的，找到元素后和冒号前匹配，不一致就找不到。

伪元素选择器

伪元素可以帮助我们利用css创建标签元素，而不需要要html，从而简化html结构

  • element::before
  • element::after
  • before和after创建一个行内元素
  • 新创建的元素在文档树内找不到，所以叫做伪元素
  • 必须有content属性
  • before在父元素前，after在后
  • 权重为1
