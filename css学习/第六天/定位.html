定位: 将盒子定在某一个位置
定位 = 定位模式 + 边偏移

定位模式

定位模式决定元素的定位方式，通过position属性去确定。

  • static 静态定位
  • relative 相对定位
  • absolute 绝对定位
  • fixed 固定定位

边偏移

定位的盒子，相对于某个元素(定位模式有关)移动的距离。有top bottom left right4个属性

静态定位(默认)

元素遵循标准流，没有边偏移，很少用

相对定位

  • 元素以之前的位置为参照
  • 遵循标准流
  • 应用中，给绝对定位当爹

绝对定位

  • 元素以祖先元素为参照，如果没有祖辈元素或者祖辈元素没有设置定位，就以浏览器为参照
  • 如果祖辈元素有定位，就以最近一个有定位的元素为参照
  • 脱离标准流，不占位置

日常应用： 子绝父相

固定定位

  • 固定于可视窗口
  • 跟父元素没关系
  • 不随滚动条滚动
  • 脱离标准流

粘性定位 sticky

  • 以可视窗口为参照
  • 占位
  • 必须有 top bottom left right 其中一个
  • 相对定位 => 绝对定位
